Fast Mush Game

 

Info

It is a unfinished playable demo, that served as my try on making platformers, using Godot and Game Engines as a whole. The movement here is not perfect, but I am actually really proud of it, as I achieved here some pretty advanced functionality like wall jumping, dashing or hurt animation. The Demo contains a single level with one type of enemy, coins and some little takes tilemap functionality like fading outer walls when you come into building and openable paths.

Due to how much fun I had doing this type of game, and using Godot as a whole i recently decided to open the project again, move all the code to the trash and recreate it from the very start. As for now, I directed that platformer towards the speed based one. More info about the progress will come if the project will survive the early stage.

 

What it has:

  • Pretty advanced movement system
  • Pretty characters

Tools used:

  • Godot

 

What I learned from it:

  • Animations in Game Engines (as Unity technically has the same thing)
  • Fundamentals of Godot Engine
  • Events/Signals
  • Tilemaps
  • Linear Algebra

My biggest mistakes:

  • I've put most of the physics on one update function with ifs to be called each frame
